Apple India Homepage Clone

This project is a HTML and CSS clone of the Apple India homepage (2025). It replicates the look and feel of the official apple.com/in website using only front-end code.

 Features

- Responsive navigation bar with links to core Apple product categories
- Hero product banners for iPhone 17 Pro and iPhone 17 MagicChromatic with modern styling
- Highlight sections for MacBook Pro and a grid promotion section for 6 Apple products:
  - Watch Series 10
  - iPad Pro
  - AirPods Pro (3rd Gen)
  - iPhone Air
  - MacBook Air M5
  - Apple Trade In
- Authentic footer with legal info, store locator, and site map links
- Fully responsive layout with CSS Flexbox and Grid for desktop and mobile
- No JavaScript, focusing purely on semantic HTML and modern CSS techniques
- Educational project for learning frontend development and UI replication

Usage

Clone or download this repository and open index.html in a web browser.

Replace placeholder image paths in the HTML src attributes with your own images or Apple official images for full visual fidelity.
